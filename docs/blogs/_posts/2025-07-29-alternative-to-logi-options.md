*July 29, 2025*

Alternatives to Logitech's Logi Options+ software for macOS:

- [Mac Mouse Fix](https://macmousefix.com/) — A lightweight utility to customize mouse behavior on macOS
- [SteerMouse](https://plentycom.jp/en/steermouse/) — Advanced mouse driver for macOS with customizable buttons and acceleration
- [USB Overdrive](https://www.usboverdrive.com/index.php/download/) — Driver for USB input devices with button mapping capabilities

### AWS Kinesis

```bash
aws kinesis describe-stream --stream-name <name> --region us-east-2
aws kinesis put-record --stream-name <name> --partition-key 1 --data $(echo -n "1" | base64) --region us-east-2
aws kinesis get-shard-iterator --stream-name <name> --shard-id <id> --shard-iterator-type TRIM_HORIZON --region us-east-2
aws kinesis get-records --shard-iterator <iterator>
```

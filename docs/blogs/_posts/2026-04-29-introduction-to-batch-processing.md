## Batch Processing vs Streaming Processing

| Feature | Batch Processing | Streaming Processing |
| --- | --- | --- |
| Description | Batch flows are important. Processing data at regular intervals. | Processing the data as and when it comes with minimal delay |
| Frequency | Hourly, Daily, Weekly, Monthly, Yearly | Continuous (real-time) |
| Advantages | Easy to manage, Retry, Easy to scale, Processing speed | Faster access of data |
| Disadvantages | Delay in execution and running of the jobs | When large quantities of data need to be analyzed, streaming is not a viable option |

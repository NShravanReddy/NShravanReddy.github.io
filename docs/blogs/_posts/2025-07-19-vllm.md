*July 19, 2025*

VLLM is a high-performance inference engine for LLMs.

### Serving a Model

```bash
VLLM_USE_TRITON_FLASH_ATTN=0 \
vllm serve Qwen/Qwen3-30B-A3B \
  --served-model-name Qwen3-30B-A3B \
  --api-key abc-123 \
  --port 8000 \
  --enable-auto-tool-choice \
  --tool-call-parser hermes \
  --trust-remote-code
```

### Tool Calling

- VLLM supports tool calling with `--enable-auto-tool-choice` and `--tool-call-parser`
- Uses GPGPU and SIMD architecture for efficient inference
- Transformer matrix multiplication as core operation

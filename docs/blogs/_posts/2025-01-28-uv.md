*January 28, 2025*

[UV](https://docs.astral.sh/uv/getting-started/installation/#standalone-installer) is a fast Python package manager.

### Installation

```bash
pip install uv
```

### Usage

```bash
# Initialize a project
uv init

# Create a virtual environment named satvick
uv venv satvick

# Activate the environment
source /bin/activate

# Or create a default virtual environment
uv venv

# Install packages
uv pip install ipykernel

# Install PyTorch (CPU version)
uv pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu

# Install dependencies from pyproject.toml
uv pip install -r pyproject.toml

# Activate a specific venv
source ./deepseek/bin/activate

*January 28, 2025*

Currently from few years I am Creating the virtual environments and managing using uv

Let me give you some examples how I am using uv

## Example 1: Install packages using uv pip

```
pip install uv
```

```
uv init
```

```
uv venv satvick
```

```
source satvick/bin/activate
```

```
uv pip install ipykernel
```

```
uv pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
```

---

## Example 2: Install packages using uv add

```
pip install uv
```

```
uv init
```

```
uv venv satvick
```

```
source satvick/bin/activate
```

```
uv add ipykernel
```

---

## Example 3: Install an existing project's dependencies

```
source deepseek/bin/activate
```

```
uv sync
```

If you have a traditional requirements.txt, then use:

```
uv pip install -r pyproject.toml
```
